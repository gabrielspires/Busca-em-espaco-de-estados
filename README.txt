Para executar esse trabalho use o comando da seguinte forma:

./busca.sh -algoritmo <algoritmo> -entrada <arquivo_de_entrada>

<algoritmo> = bfs, dfs, ids ou a_star

Exemplo:
./busca.sh -algoritmo bfs -entrada Entries/input16x8-3.txt
./busca.sh -algoritmo dfs -entrada Entries/input16x8-3.txt
./busca.sh -algoritmo ids -entrada Entries/input16x8-3.txt
./busca.sh -algoritmo a_star -entrada Entries/input16x8-3.txt
